适配器模式
可以把一个类的接口变换成客户端所期待的另一种接口，使得原本因接口不匹配而无法在一起工作的两个类可以一起工作。

适配器模式的用途
比方说我有一个台灯，其插头是标准的两相的交流点插头，即阳极、阴极。我旅游到了一个地方想用自己的台灯，但发现旅馆里面只有三
相的插头，即在阳极、阴极的基础上还多了一个地极。这个时候一个两相到三相的转换器（适配器）就能解决这个问题。

类适配器
适配器模式分两种，类适配器和对象适配器。

类适配器把适配的类的API转换成目标类的API，类适配器涉及的角色有：
1、目标角色
    这是所期待的到的接口

2、源角色
    现在需要适配的接口

3、适配器角色
    模式的核心，适配器把源接口转换成目标接口
